# Spring Boot整合实战
本项目主要内容为：
* 搭建第一个spring boot项目 
* Mybatis 
* Mybatis plus
* swagger2
* Redis
* WebSocket
* Ribbit mq
* Shiro
* Thymeleaf
* Aop
* Spring Cache
## 开发环境
* JDK 13
* Idea 2020.1
* Mysql 8.0
* Spring Boot 2.4.0
* Navicat 12
* Apache Maven 3.6.3
* Redis-x64 3.2.100
* Redis Desktop Manager 2020.4
